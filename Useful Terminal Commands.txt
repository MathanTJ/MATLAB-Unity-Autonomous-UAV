ROS2 nodes:
ros2 run ros_tcp_endpoint default_server_endpoint --ros-args -p ROS_IP:=<your IP>

SLAM Toolbox
ros2 run unity_slam_bringup odom_to_tf_broadcaster
ros2 launch unity_slam_bringup unity_slam_bringup.launch.py

Nav2 Stack:
ros2 launch nav2_bringup navigation_launch.py \ use_sim_time:=false
ros2 launch nav2_bringup slam_launch.py

RVIZ Nav2:
ros2 launch nav2_bringup rviz_launch.py

To launch MATLAB:
/usr/local/MATLAB/R2025b/bin/matlab
To set window scale:
s = settings;
s.matlab.desktop.Zoom.PersonalValue=120

Get IP:
hostname -I

To launch ROS2 and any nodes in ros2_ws:
cd ~/ros2_ws 
source /opt/ros/kilted/setup.bash
source install/setup.bash
ros2 run ros_tcp_endpoint default_server_endpoint --ros-args -p ROS_IP:=192.168.1.153

To remove colcon build in workspace run: 
rm -r build/ install/ log/

To get ROS2 graph:
ros2 run rqt_graph rqt_graph

To run Gazebo:
gz sim
ros2 launch turtlebot3_gazebo turtlebot3_world.launch.py 

To get list of installed packages:
apt list --installed | grep <package>

ros2 param get /slam_toolbox use_sim_time
